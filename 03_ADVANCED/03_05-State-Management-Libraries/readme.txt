Instalación de Redux: Primero, instala Redux en tu proyecto:

intalar npm desde la web

Con npm instalado, crear un proyecto de vite:

create vite@latest nombre-de-mi-app
(seleccionar el empaquetado deseados: React, Vanilla, Solid, Javascript, etc)


npm install redux react-redux

Una vez la librería esté descargada, se procede a iniciar la aplicación:

npm run dev
