React Topics - Set 1

# Topic Description Advantages Real-World Scope
1 Hello World in React Basic setup of a React app using `ReactDOM.render()` or `createRoot()` to mount an element into the DOM. - Quick setup
- Foundation for all React applications
- Helps beginners understand component-based architecture		 Used to kickstart any React project and ensure dependencies are correctly installed.
2 JSX Syntax JSX (JavaScript XML) allows writing HTML-like code inside JavaScript, which React transpiles into `React.createElement()` calls. - Improves code readability
- Allows embedding expressions with `{}`
- Prevents injection attacks by escaping values		 Used in every React component to define UI structures in a clean and declarative way.
3 Functional Components Stateless components written as JavaScript functions that return JSX. - Simple and readable
- Easier to test and reuse
- Works well with Hooks		 Ideal for UI components like buttons, headers, and reusable widgets.
4 Class Components ES6 class-based components extending `React.Component`, allowing lifecycle methods. - Supports lifecycle methods like `componentDidMount()`
- Can handle state without Hooks
- Useful for complex logic		 Common in legacy React applications before the introduction of Hooks.
5 Props in Functional Components Props (short for properties) are read-only arguments passed from parent to child components. - Enables component reusability
- Helps maintain a unidirectional data flow
- Simplifies UI updates		 Used for passing data like user profiles, settings, or dynamic content to components.
6 Props in Class Components Similar to functional props but accessed using `this.props`. - Supports prop validation with PropTypes
- Works with lifecycle methods
- Enables structured data passing		 Useful in older applications that require class-based components with state.
7 State in Class Components Component state is stored in `this.state` and updated with `this.setState()`. - Allows dynamic UI updates
- Supports complex state management
- Enables component reactivity		 Common in older React applications where state management was required before Hooks.
8 useState Hook A Hook that enables state management inside functional components. - Simplifies state handling
- Reduces the need for class components
- Improves code maintainability		 Used in almost all modern React applications to handle UI state.
9 Handling Click Events Using `onClick` event handlers to trigger functions when a user interacts with an element. - Enables interactive UIs
- Improves user experience
- Allows dynamic event handling		 Common in buttons, links, and user interactions like forms and modals.
10 Conditional Rendering Using JavaScript expressions (`if`, ternary operators) to show or hide UI elements dynamically. - Improves performance by rendering only necessary elements
- Reduces unnecessary DOM updates
- Allows flexible UI logic		 Used in dashboards, login pages, and error messages where UI visibility depends on state.
11 Lists and Keys Rendering lists using `.map()` and assigning unique keys for optimized performance. - Improves rendering efficiency
- Prevents unnecessary re-renders
- Enhances performance in dynamic lists		 Used in displaying lists like product catalogs, user comments, and dynamic menus.
12 Forms and Controlled Components Handling form inputs where the React state controls the values. - Provides real-time validation
- Enables centralized state management
- Improves form behavior consistency		 Used in login forms, user registrations, and data input fields.
13 Uncontrolled Components Forms that rely on the DOM instead of React state for managing values. - Simpler setup for non-interactive forms
- Reduces state management complexity
- Works well with third-party libraries		 Useful in file uploads, third-party form integrations, and legacy form handling.
14 Component Lifecycle Methods Class component lifecycle methods like `componentDidMount()`, `shouldComponentUpdate()`, and `componentWillUnmount()`. - Allows precise control over component behavior
- Optimizes performance by preventing unnecessary renders
- Helps in managing side effects		 Commonly used for API calls, subscriptions, and resource cleanup.
15 useEffect Hook Hook for handling side effects in functional components, replacing lifecycle methods. - Reduces boilerplate code
- Allows better dependency management
- Handles component mounting, updating, and unmounting		 Used for API calls, setting up subscriptions, and handling cleanup tasks.
16 React Fragments Used to group multiple elements without adding extra DOM nodes. - Improves DOM performance
- Reduces unnecessary HTML elements
- Keeps the structure clean		 Useful in rendering multiple sibling elements without additional `div` wrappers.
17 React Portals Renders a component outside its parent DOM hierarchy using `ReactDOM.createPortal()`. - Allows rendering modal dialogs, tooltips, and popups without affecting parent styles
- Improves accessibility and structure
- Helps manage overlays effectively		 Commonly used for modals, popups, and notifications.
18 Handling Events Binding event handlers in JSX and passing event objects to functions. - Enables dynamic user interactions
- Provides control over event propagation

- Helps in controlling component behavior efficiently		 Used in form handling, toggles, counters, and UI updates.
